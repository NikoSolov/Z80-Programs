# Описание

Собранная простейшая модель компьютера на процессоре `Z80` 

- Процессор `z80`
- ПЗУ 64кб (используется 32 кб)
- ОЗУ 32 кб
- Устройство Ввода/Вывода `КР580ВВ55`,
- Тактовый генератора: `Arduino`
